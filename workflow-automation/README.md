# Supply Chain Automation Tools  
## Outlook Email Automation (AutoHotkey)

### 📌 Overview
This project automates repetitive Outlook email workflows in supply chain operations using AutoHotkey.

It is designed to reduce manual workload, improve efficiency, and standardize communication processes in logistics and customs environments.

---

### ⚙️ Features
- Generate booking / arrival notification emails automatically  
- Reply-all with ETA updates  
- Quick "well received" email responses  
- Keyboard navigation shortcuts in Outlook  

---

### 🚀 Business Impact
- Reduced manual email writing time by ~70%  
- Improved response speed in high-volume logistics operations  
- Standardized communication templates across workflows  

---

### ⌨️ Hotkeys
- Shift + W → Move up  
- Shift + S → Move down  
- Alt + N → New booking email  
- Alt + E → ETA reply  
- Alt + W → Well received reply  

---

### 🛠️ Tech Stack
- AutoHotkey  
- Outlook COM Automation  
- HTML Email Templates  

---

### ⚠️ Notes
This script is designed for Outlook on Windows environment.
